# AI Assistant Jarvis 🤖

This is an Artificial Intelligence assistant project named Jarvis. I've been working on that can handle chats and help automate tasks.
It's built using Next.js, and I'm really excited to keep developing it further

## Features
* Voice Commands
* Smart Answers
* Task Management

## How to Run
1. Clone this repository:
   git clone https://github.com/RABIWARSI/ai-assistant-jarvis.git
2. Run the main file:
   python main.py
